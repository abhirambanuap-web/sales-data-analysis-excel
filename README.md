# 📊 Sales Analysis Dashboard

## 📌 Project Overview

The Sales Analysis Dashboard provides a comprehensive view of business performance by analyzing sales, profit, quantity, and margin across multiple years. This project focuses on transforming raw sales data into meaningful insights using interactive reporting techniques in Microsoft Excel.

The dashboard enables stakeholders to monitor key performance indicators (KPIs), identify trends, and make data-driven decisions to improve business outcomes.

---

## 🎯 Objectives

* Analyze multi-year sales data to track business performance
* Build an interactive dashboard for dynamic reporting
* Identify key trends across regions, products, and customer segments
* Support decision-making with actionable insights

---

## 🛠️ Tools & Technologies

* **Microsoft Excel**
* Pivot Tables & Pivot Charts
* Data Cleaning & Transformation
* Data Visualization

---

## 📂 Dataset

The dataset contains:

* Sales transactions (2020–2023)
* Product categories and segments
* Regional and city-level data
* Profit, quantity, and margin metrics
